# Compiler

Mini compiler that can compile .cpp source code into machine language and turn it into an
executable.

## Usage

On Command Line or a Terminal type in:

`make`

you can also use:

`compile.bat`

if it still does not work, type in:

`g++ main.cpp -o main`

to compile the file.

For executing, type in

`./main source.mcpp`

or you can try

`main.exe source.mcpp`

or

`./main.exe source.mcpp`
